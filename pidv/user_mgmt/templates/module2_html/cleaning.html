Data Cleaning

  • The data can have many irrelevant and missing parts. To handle this part, data cleaning is done. It involves handling of missing data, noisy data etc.

  • How would you like to handle the cleaning of data:
    • Handling Missing Values:
    • DROP ROWS/COLS OF MISSING DATA

      Ignore/Drop the tuples/columns: This approach is suitable only when the dataset we have is quite large and multiple values are missing within a tuple(row)/column.

    • Forward/Backward Fill

      *A forward fill finds any missing data, and sweeps "forward" the last known previous value.

    • Replace Missing Data

      Replace a missing data with certain value by selecting perticular column.

  • Duplicate row elimination

    It determines whether data contains duplicate representations of the same entity. If so then it merges them together.

  • Data transformation

    Data transformation allows the mapping of the data from its given format into the format expected by the appropriate application. This includes value conversions or translation functions, as well as normalizing numeric values to conform to minimum and maximum values.

  • * Take note how your data is ordered by default. Some date-time data may be ordered with oldest date first, or newest or based on some column. Check the table by Open Source File to confirm how your data is ordered! Otherwise sort it from Sort tab based on required column.
